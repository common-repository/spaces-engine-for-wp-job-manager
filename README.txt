Spaces Engine for WP Job Manager allows your Space administrators to post jobs inside their Spaces.

== Description ==

Spaces Engine for WP Job Manager is a must-have extension to connect your SPaces with the best WordPress job board plugin.

For a full list of features, documentation, screenshots and videos, head to the [plugin page](https://spacesengine.com).

== Installation ==

1. Go to WordPress Dashboard->Plugins->Add New
2. Search for Spaces Engine for WP Job Manager using search option
3. Find the plugin and click Install Now button
4. After installation, click on Activate Plugin link to activate the plugin.
